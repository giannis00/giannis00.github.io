5CM512 — Communication and Security Protocols

Cryptographic fundamentals, security protocols, and hands-on protocol analysis.

Instructor: Ioannis Tsioulis

Term: Level 5 (UG)

Location: Mediterranean College, Athens

Overview

Communication and security protocols: cryptography, symmetric/asymmetric encryption, network protocol analysis, PKI, and secure communications.

Taught in: 2024-2025.
