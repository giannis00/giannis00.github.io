4CM508 — Computer Systems, Data Structures, and Data Management

Fundamental data structures and algorithms with an introduction to database systems.

Instructor: Ioannis Tsioulis

Term: Level 4 (UG)

Location: Mediterranean College, Athens

Overview

Fundamental data structures and algorithms with an introduction to database systems, covering C++ programming, classical algorithms, and SQL.

Taught in: 2024-2025.
