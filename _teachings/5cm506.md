## Overview

This module covers the full stack of data-driven application development: relational database design, SQL, normalisation, and integration with web technologies. Students develop hands-on skills in SQL querying, T-SQL scripting, Microsoft Access, and server-side web development with PHP and REST APIs.

## Topics

- The relational model and entity-relationship design
- Database design and normalisation (1NF, 2NF, 3NF)
- SQL: single-table queries, joins, outer joins, and aggregates
- Working with NULLs and advanced query features
- T-SQL: cursors and JSON
- Database programming
- Web integration: HTML, PHP, and REST APIs
- Microsoft Access: forms, reports, and switchboards
