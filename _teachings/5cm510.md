## Overview

This module examines modern network architectures and emerging technologies, from peer-to-peer and IoT systems through software-defined networking, network function virtualisation, and fifth and sixth generation mobile networks. Security principles are integrated throughout.

## Topics

- Peer-to-peer network architectures
- Internet of Things (IoT) systems
- Software-Defined Networking (SDN)
- Network Function Virtualisation (NFV)
- 5G and 6G mobile networks
- Quantum networking fundamentals
- Network security
