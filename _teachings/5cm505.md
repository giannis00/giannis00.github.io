## Overview

This module introduces systematic approaches to software development, from requirements capture through design, implementation, testing, and maintenance. Students apply UML modelling techniques and study established methodologies for managing the full software lifecycle.

## Topics

- Requirements analysis and specification
- UML: use case, class, and sequence diagrams
- Software design patterns
- Software testing strategies (unit, integration, system)
- Software maintenance and evolution
- Agile and plan-driven development methodologies
