## Overview

This module builds programming fundamentals in C++ alongside a study of classical data structures and algorithms. Students progress from core language concepts and control flow through arrays, linked lists, trees, and standard searching and sorting algorithms. The module concludes with an introduction to database management and SQL.

## Topics

- Data types, variables, and control flow
- Arrays and dynamic data structures
- Searching algorithms: linear and binary search
- Sorting algorithms
- Linked lists and trees
- Introduction to databases and the relational model
- SQL: queries, joins, and aggregation
