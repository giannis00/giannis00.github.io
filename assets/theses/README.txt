# Student theses

Drop the selected student dissertation PDFs here, e.g.:
- 2025-msc-student-name-title.pdf
- 2024-bsc-student-name-title.pdf

Then link them from _pages/supervision.md as /assets/theses/<filename>.
Tip: get the student's consent before publishing their full thesis, and consider linking an abstract page rather than the full PDF if they prefer.
